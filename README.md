Projeto de Sistemas de Informação e Enganharia de Software I do curso de Análise e Desenvolvimento de Sistemas da Fatec Cruzeiro Prof: Waldomiro May. O projeto é um sistema de remoção de produtos baseado na nota média recebida pelo usuário, em que se o produto tiver uma aprovação menor do que 50%, o produto é removido e desospedado do sistema/site.

Nossa empresa baseia-se em um site de varejo e-commerce para venda de produtos gerais não-especificados e fomos chamados para desenvolver um sistema que pudesse se basear nas avaliações das pessoas sobre os produtos para os deleta-los caso suas aprovações não fossem as desejadas, que no caso deste projeto é 50%, dessa maneira o sistema deveria funcionar de maneira a tirar a média das avaliações em geral e desospedar os produtos que não tiverem a meta desejada, tudo isso deve ser feito de mês em mês, período especificado pela empresa que nos pediu o sistema, fazendo assim o sistema executar suas funções no primeiro dia de cada mês, analisando os produtos e decidindo se ficarão hospedados ou não, também separamos os produtos que não serão desospedados em categorias, sendo elas bom(2,5-3,5), muito bom(3,5-4,5) e excelente(4,5-5,0), dessa maneira o sistema pode categorizar produtos que não foram deletados para melhor reconhecimento de suas notas.

Para nossas tecnologias nós usamos o figma para podermos criar telas navegáveis de como deve funcionar as avaliações e como as categorias ficam mostradas, tanto a média como a desospedagem e categorização exemplificados nas telas e também fizemos uso do java para desenvolver as funções que o sistema utilizará para tirar a média das avaliações dadas, decidir se o produto será desospedado ou não e categorizar o produto caso ele n tenha sido deletado, para todas essas funções foram criados casos de teste de maneira a poder verificar a funcionalidade e confiança das funções, no total foram 11 casos de teste que em média houveram 2 para cada função do sistema de maneira a demonstrar a confiança do sistema.
